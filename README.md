### 📦 Featured Projects

#### [req-res-crypto](https://github.com/wilbur-yu/req-res-crypto-core) — Anonymous AEAD Encryption Protocol

End-to-end request/response encryption with forward secrecy. Four packages, one protocol.

| Package | Description |
|:--- |:--- |
| [**req-res-crypto-core**](https://github.com/wilbur-yu/req-res-crypto-core) | Core PHP library, zero framework dependency |
| [**req-res-crypto-hyperf**](https://github.com/wilbur-yu/req-res-crypto-hyperf) | Hyperf adapter — PSR-15 middleware, DB key rotation |
| [**req-res-crypto-laravel**](https://github.com/wilbur-yu/req-res-crypto-laravel) | Laravel adapter — middleware, Facade, annotation-driven |
| [**req-res-crypto-js**](https://github.com/wilbur-yu/req-res-crypto-js) | Browser client — axios/fetch transparent crypto |

> X25519 ECDH key exchange + XChaCha20-Poly1305 AEAD. Each request generates ephemeral key pairs (~0.05ms), zero signing overhead.
